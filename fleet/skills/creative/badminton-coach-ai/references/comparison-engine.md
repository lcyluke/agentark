# 比较引擎参考文档 — comparison_engine.py

## 架构

```
用户骨架JSON (33 landmarks, MediaPipe格式)
        │
        ▼
load_user_skeleton() ──→ 15 关键点格式 (关键羽毛球关节点)
        │
        ▼
dtw_align() ──→ 角度空间 DTW (10维特征，比坐标DTW更鲁棒)
        │        对齐路径: 104对 (50帧用户 × 120帧基准)
        ▼
compute_angle_deviation() ──→ 每关节角度偏差统计
        │                     (mean_diff, max_diff, std_diff, user_avg, pro_avg)
        ▼
generate_report() ──→ 中文纠错话术 (3级严重度: 🔴>20° 🟡12-20° 🟢<12°)
        │
        ▼
_compute_overall_score() ──→ 综合评分 (加权: 右肘0.25 > 右肩0.20 > 膝0.15 > 腰0.10 > 左侧0.05)
        │
        ▼
_generate_training_plan() ──→ 7天训练计划 (按优先级排序)
```

| 关节 | 索引(15点制) | 权重 | 说明 |
|:-----|:-----------:|:----:|:-----|
| 右肘 | (4,5,7) | 0.25 | 持拍手肘关节角度 |
| 左肘 | (3,6,8) | 0.05 | 非持拍手(参考) |
| 右肩 | (5,4,9) | 0.20 | 持拍手与躯干夹角 |

| 左膝 | (9,12,14) | 0.10 | 支撑腿稳定性 |
| 右髋 | (4,9,11) | 0.05 | 重心高度 |

## 纠错话术模板

存放在 CORRECTION_TEMPLATES 字典中，每条包含:
- too_bent / too_straight → 对应不同方向偏差
- 变量 {val} {ref} {diff} 自动填充
- 包含具体练习建议

## 综合评分标准

| 等级 | 分数范围 | 标签 | 颜色 |
|:----:|:--------:|:----|:----:|
| A | ≥85 | 优秀 | 绿 |
| B | ≥70 | 良好 | 蓝 |
| C | ≥50 | 需加强 | 黄 |
| D | <50 | 需改进 | 红 |

## 中文话术完整性

覆盖的关节和对应练习:
- 右肘: 太曲→对墙扔球/太直→屈伸练习
- 右肩: 太低→架拍练习/太高→侧身对镜
- 右膝: 弯曲不足→半蹲跳/过直→落地缓冲
- 腰部扭转: 不足→转体练习/过度→控制幅度
- 其他关节: 通用模板 (对照镜子练习)

## 产品验证 (2026-06-02)

用户明确判定: MimicMotion生成的"你做标准动作"视频不是付费核心。
**真正的付费功能**是 上传自己视频 → 比对 → 标出差距 → 给训练计划。
```
你上传自己打球视频
    ↓
AI 提取你的骨骼动作
    ↓
vs 职业选手标准动作逐帧比对
    ↓
标出差距（手腕角度差15°、击球点低20cm、转体不充分）
    ↓
生成针对性拆解练习计划
    ↓
你练完再拍 → 再比对 → 看进步
```

## 小程序架构

`pages/compare/compare` 页面:
- 6维指标对比图 (右肘/右肩/右膝/腰部/击球高度/重心稳定)
- Top-3 问题卡片 (中文+严重度+练习建议)
- 7天训练计划 (横向滚动)
- 总体匹配度评分 (颜色编码)

## API端点

| 方法 | 路径 | 功能 |
|:----:|:-----|:-----|
| POST | /api/compare | 上传视频 → MediaPipe骨骼 → 比对 |
| GET | /api/compare/from-file?skel_path=&action_type= | 直接文件路径比对(测试用) |
| POST | /api/compare/skeleton | JSON骨架比对 |
| GET | /api/compare/benchmarks | 列出8个可用基准 |

## 基准文件

位置: `badminton-label-system/data/benchmarks/`
格式: NPY (T, 15, 3)
已存在8个: smash_L6, clear_L5, drop_L6, net_L6, footwork_L5, feint_L5, def_L6, amateur_L6
文件命名: `{action_type}_L{level}.npy` — 自动选最高级别
