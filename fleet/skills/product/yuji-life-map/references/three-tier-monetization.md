## 等级1：免费3个月（获客+养习惯）

| 设计维度 | 说明 |
|---------|------|
| ⏱️ **限时策略** | 不是永久免费，3个月后自动降级为「基础版」 |
| 🧠 **心理学** | 66天习惯养成周期，3个月让用户离不开 |
| 🏸 **打球频率** | 深圳球友平均每周2次 → 3个月≈24次打开羽迹 |
| 📸 **功能限制** | 每次5张照，每次只识别1种动作（吊胃口） |
| 🃏 **社交裂变** | 趣味角色分享卡片无限发（拉朋友来测） |
| 📈 **数据积累** | 3个月收集动作偏好、常去场馆、进步数据 |
| 💬 **到期转化话术** | "你3个月测了72次击球，杀球从63→71分！但你还没测高远吊球挑球网前→升级解锁全面评估！" |

### 核心付费点（杀手功能）

**搭档兼容度分析** — 用户A测了发现是"后场重炮型"，自然想拉搭档B来测——这是一个天然社交裂变+付费驱动。

| 项目 | 价格 |

### 设备方案

**方案A（零成本启动）**：
- 用户自己的两部手机（正面+侧面）→ AI 3D融合
- 精度比单机高50%以上

**方案B（投资型设备包 ~¥3000/套）**：
- 2K运动相机×2 + 三脚架 + 遥控同步
- 可选配置：测速雷达（¥500）
- 回本：8次专业评估即回本

### 服务流程
```
下单 → 匹配评估师 → 现场架设双机位
→ 引导完成7项标准化测试
→ AI分析 + 评估师复核
→ 出L1-L7等级 + 纸质精装证书
→ 数据进入羽迹等级数据库（长期追踪）
```

## 🏪 评估师生态（平台化护城河）

### 评估师来源
- 羽毛球教练
- 体育院校学生（兼职）
- 退役运动员
- 资深球友（L5+，经羽迹培训认证）

### 商业模型

```
评估师注册费: ¥499/年 或 ¥99/月
认证培训: ¥999（可选）

评估师接单: ¥399/次
羽迹抽成 20%: ¥79.8
评估师到手: ¥319.2/次

如果一个评估师每月接10单 → 月收入 ¥3,190
年付注册费¥499 ÷ 12 = ¥42/月
月净收入 ≈ ¥3,148

羽迹平台年收入估算 (深圳试点50评估师):
  评估师注册: ¥25,000
  评估抽成: ¥383,000
  场地返佣: ¥120,000
  Σ ≈ ¥528,000
```

## 评估师注册系统设计（未来平台功能）

```
评估师注册流程
1. 提交资料（资质/证书/球龄）
2. 羽迹审核（AI+人工）
3. 线上培训+考核
4. 缴纳注册费
5. 开通接单权限
6. 可查看附近预约订单
7. 接单→到场→完成→平台抽成→到账

提成结构
  专业评估 ¥399
  ├── 羽迹平台 20% = ¥79.8
  ├── 评估师 80% = ¥319.2
  └── 场地返佣 ¥20-50（可选）

额外收入
  ├── 私人教练课程（羽迹平台引荐，抽成10%）
  ├── 装备推荐（品牌合作返佣）
  └── 俱乐部训练营（羽迹招募，分成30%）
```

## 用户典型路径

| 用户类型 | 路径 | 羽迹收益 |
|---------|------|---------|
| 🧑‍💼 普通球友(80%) | 免费→偶尔用→分享→免费 | 流量+传播 |
| 🏋️ 认真球友(15%) | 免费→2个月→付费¥29/月→持续用 | ¥29×12=¥348/年 |
| 🏆 发烧友(5%) | 免费→付费→¥399专业评估→升级评估师 | ¥29+¥399+¥499/年 |

## 关键原则

- `业余评估先跑通再收费` — P0/P1免费，P2才上付费墙
- `先方案A手机拍摄再方案B设备包` — 零成本验证需求
- `评估师生态是终极护城河` — 抄代码容易，抄评估师网络难
